# socket.io-unit

Unit testing

## Install

`yarn add socket.io-unit`

## How to use

    const SocketioUnit = require
	('socket.io-unit');

## Tests

[test/BasicTest.js](test/BasicTest.js)

`SOCKET_PORT=x`

first start the test server with `yarn run server` then run `yarn run test`.

Use `yarn run lint` to run the linter.
